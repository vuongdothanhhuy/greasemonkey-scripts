    What is this?

    This is my personal collection of greasemonkey scripts. I initially developed them for my own personal use.

    How to use?

    Most, if not all, of these scripts are also hosted at https://openuserjs.org/users/vuongdothanhhuy/scripts, so you can visit it, choose and install the script you like, with one click. Of course you will need to have Greasemonkey plugin (if you use Firefox), or Tampermonkey extension (if you use Chrome/Chromium) installed first.
